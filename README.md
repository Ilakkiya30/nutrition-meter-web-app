# 🥗 Nutrition Meter Web App

A full-stack nutrition tracking web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

## Features

* Add and delete food entries
* Track calories, protein, carbohydrates, and fat
* Real-time nutrition dashboard
* Daily calorie limit monitoring
* Responsive user interface
* MySQL database integration

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* MySQL
